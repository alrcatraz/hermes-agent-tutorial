# 引言

Hermes Agent 是一个功能强大的开源 AI 智能体框架。你可能听说过 Claude Code、OpenAI Codex 或 Cursor——这些工具让 AI 直接在你的终端里操作代码和系统环境。Hermes Agent 和它们属于同一类别，但它的设计目标是走得更远：**不仅能在终端里工作，还能通过 Telegram、Matrix 等平台与你互动，拥有跨会话记忆、技能积累和多模型协作能力。**

本教程从**零基础**开始，逐步带你走过安装、配置、到高级调优的完整路径，分为三卷共24章：

- **第一卷（第1～6章）** — 基础配置：安装、首次启动、Gateway 接入、工作原则
- **第二卷（第7～12章）** — 进阶设置：多模型协作、外部记忆、SearXNG、Agent 定制
- **第三卷（第13～24章）** — 高级实战：Astra 生态总览、Astra Hub、凭据管理、Agentic Harness、知识库、Skill 体系、上下文锚定、浏览器自动化、Office 工具、SRE 运维

无论你是第一次接触 AI Agent 的新手，还是想深度定制 Hermes 的老手，这份教程都有你需要的资料。
